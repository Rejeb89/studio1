# **App Name**: ImageScribe

## Core Features:

- Image Upload: Allow users to upload an image from their device or paste an image URL.
- Text Extraction via OCR: Extract text from the uploaded image using Optical Character Recognition (OCR) technology.
- AI-Powered Formatting Tool: Use generative AI to format and improve extracted text; provide clean output by, when appropriate, correcting spelling, improving grammar and creating paragraphs.
- Download as Word Document: Automatically format the extracted text into a .docx file and provide a download link for the user.
- Download button: Enable the download functionality after extraction

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for trust and readability.
- Background color: Light gray (#F0F4F8) for a clean and professional look.
- Accent color: Soft green (#A5D6A7) to highlight the download CTA.
- Body and headline font: 'Inter' sans-serif for a clean and readable interface.
- Use minimalist icons for actions like upload and download.
- Simple and intuitive layout with clear instructions for each step.
- Subtle loading animations during the text extraction process.
